# Lotus_And_Friend
Scripts supporting the analysis of circRNA identification and characterization from an experiment using Lotus japonicus and Mesorhizobium loti


*Scripts* contains all R scripts used for data analysis. 
*Data* contains those datasets that are practical to store on github alongside scripts, mosty intermediary datasets and endpoints of analyses. Additional data is available on the SRA: or via contacting the authors; see: 
*Figures* contains some of the figures generated in R which are mostly components or full panel figures used in the publication.
